## Steps to Setup:-

1. Download the project at local machine.
2. Open project in VS Code.
3. For frontend(Terminal 1):
    1. `cd frontend`
    2. `npm install`
    3. `npm start`
4. for backend(Terminal 2):
    1. `cd backend`
    2. `Scripts\activate`
    3. `uvicorn main:app --reload`
5. May use sample images in backend\images folder.
